Calendar

To help support your professional development, we are happy to share the following skills workshops taking place this Fall, hosted by the NYU Data Services and NYU HPC teams. These specific workshops have been highlighted as these have been identified as skills that are very helpful for a statistics-related career. For this semester, these include Python, SQL, Spark, and Tableau. We encourage you to take advantage of these services and sign up for any that are of interest.

Please take note of modality (virtual/in-person), pre-requisites, and any software or prep needed ahead of any of these workshops. Please reach out to the appropriate team (Data Services, HPC) with any questions on the workshops.

Intro to Spark

Time: Wednesday October 6th, 2021 3pm - 5pm

  • Brief overview of the Spark ecosystem
  • RDD, transformation and action
  • Running spark-shell, pyspark
  • Compiling Java code with Maven
  • Spark-submit
  • Spark SQL
  • Accessing Hive database in Spark

Data Visualization with Tableau

Registrations open at 2:00pm Wednesday, September 29, 2021

Time: Wednesday October 13th, 2021 2pm - 4pm

  • Understanding of basic principles of data visualization
  • Tableau interface
  • Importing data into Tableau
  • Exploring data and types of data
  • Creating tables, charts, scatter plots, histograms, etc.
  • Creating interactive dashboards and stories
  • Mapping data with Tableau

Introduction to Python

Registrations open Monday October 4th, 2021 8pm

Time: Monday October 18th, 2021 8pm - 11pm

  • Python interface
  • Data types (integers, floating point numbers, strings, booleans, dictionaries, lists)
  • Indexing data structures
  • Conditional statements and logical operations
  • Loops
  • Functions
  • Putting it all together in a basic web scraping example
